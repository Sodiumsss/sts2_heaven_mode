# HeavenMode

[中文](./README.md) | [English](./README.en.md)

## 简介

**Heaven** 为游戏新增了 **10 个全新的难度等级**。  
每一个难度都会加入新的挑战机制，使玩家在尖塔中面对更加严苛和多变的考验。

要解锁第一个 Heaven 难度，玩家 **必须先通关官方难度 10**。

如果你希望直接解锁全部难度，也可以手动开启：

1. 安装 Mod 后 **先启动一次游戏**。
2. 打开 Mod 文件夹。
3. 找到配置文件，将 **`unlock` 字段改为 `true`**。

完成后，所有 Heaven 难度都会被解锁。
