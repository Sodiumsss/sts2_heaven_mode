# 难度

1. 第一层ancient提供2个选项，第二层第三层只提供1个选项
2. 每通过事件，商店删一张牌，当前血量减少3点（不致死，血量最多保留到1）
3. 第二层第三层进入只恢复已损失生命值60%的血量，但是第一关开局会获得一个小血瓶
4. 从第三回合开始，每一回合都会置入一张灼烧在抽牌堆中
5. 每次战斗的第一回合，随机两张牌的费用加1；
6. 开局血量固定为36
7. 每杀死一个怪物，你受到2点伤害
8. 默认的药水栏位减为1，默认给一瓶石头药水:Potion-Shaped Rock
9. 当本回合抽空一次牌库时，抽上来的牌的耗能增加1；当第二次抽空时，耗能再增加1；依此类推；这个效果会在下一回合开始后被清除
10. 在第三层，将与3位boss决战，并且到第三位boss时，第二回合后，玩家（ ceil(n/2) ）动完一个回合后，随机挑选一张抽牌队里的卡牌烧毁；举例：玩家第二回合结束，boss开始：选择相应玩家，各自的抽牌堆内随机一张烧掉；如果没有就不烧；
